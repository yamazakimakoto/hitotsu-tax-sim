## 1. 全体式 ── 4成分を足し合わせる

家計から見た年間の差し引き（**プラス＝恩恵／マイナス＝負担**）：

```
差し引き(net) ＝ −① 非食料の増税負担
              ＋② 食料品ゼロ税率の恩恵
              ＋③ 給付付き税額控除
              ＋④ BS無償化の恩恵
              （＋⑤ 社会保険料の軽減 …参考・トグルでON/OFF）
```

`calc_household.py` の対応（厳密）：
```python
ct_net = nonfood_burden − food_benefit         # 消費税ネット（正＝負担増）
net    = −ct_net + kyufu + bs                   # ＝ −①＋②＋③＋④
```

## 3. 消費モデル ── 年収から消費支出を出す

```
D（可処分所得）   ＝ 年収 × disp        # disp＝可処分所得率（税・社会保険料 控除後の概算）
C（消費支出）     ＝ D × prop           # prop＝平均消費性向
food_total       ＝ C × f_food         # 消費支出に占める「食料（外食含む全体）」の割合
  ├ food_zero（軽減税率対象）＝ food_total ×（1 − 外食酒類率）  ← 8%→0% の恩恵を受ける
  └ food_std （酒類・外食等） ＝ food_total × 外食酒類率        ← 標準税率（負担側へ回す）
nonfood（標準税率対象）＝ C × f_nonfood ＋ food_std            # 非食料 ＋ 外食・酒類
（残り ＝ 非課税：家賃・授業料・医療自己負担・非課税品等）
```

> **【査読対応①・修正必須】食料の対象を厳密化**：軽減税率8%の対象は「飲食料品（**酒類を除く**）」で、**外食・ケータリング等は対象外**（国税庁 No.6102）。家計調査の「食料」全体をそのままゼロ税率の恩恵にすると、外食・酒類分まで恩恵に算入して**過大計上**になる。そこで `food` を **`food_zero`（軽減対象＝内食・持ち帰り）** と **`food_std`（酒類・外食・ケータリング＝標準税率）** に分割し、後者は**①標準税率の負担側**へ回す。**外食酒類率** は家計調査の傾向で年収とともに上昇（高齢は低め）：年収200→13%・350→17%・500→20%・700→23%・1000→27%・1300→30%、高齢×0.75。

## 4. ①② 消費税の増減 ── 税抜換算してから税率差を掛ける

家計調査の消費支出は**税込**とみなし、**税抜に直してから**税率差を掛ける（ここが専門家指摘の要点）。**`nonfood` は §3 のとおり 非食料＋外食・酒類（food_std）を含む**。

```
① 増税負担（非食料＋外食酒類）＝ (nonfood ÷ 1.10) × (新標準税率 − 0.10)
② 食料ゼロの恩恵（軽減対象のみ）＝ (food_zero ÷ 1.08) × (0.08 − 0.00)
消費税ネット ct_net ＝ ① − ②        # 正なら負担増
```

- 新標準税率：**0.17（既定）／0.16（切替）**。現行の標準対象（非食料・外食・酒類）は10%（`÷1.10`）。
- 軽減対象の飲食料品は現行8%（`÷1.08`）→ 0%。**ゼロ税率の恩恵は `food_zero` のみ**（外食・酒類は受けない）。
- 【事実】税率・現行区分（標準10%・軽減8%）は制度事実。【政策仮定】新標準税率16〜17%・食料ゼロは本パッケージの設計。
- **価格弾力性（消費減）は世帯別には掛けない**（マクロ§8でのみ▲1〜2兆として控除）。＝世帯試算は**静学**で、負担側にやや厳しめ（保守的）。

## 5. ③ 給付付き税額控除 ── 制度式【政策仮定】

```
給付 ＝ 基礎給付（年収逓減）
      ＋ 低所得加算（年収 ≤250万：＋0.5）
      ＋ 子ども加算（年収 ≤500万：＋1.5/人／〜700万：＋0.8/人／超：0）
      ＋ ひとり親加算（＋3.0）
      ＋ 高齢加算（高齢世帯：＋1.0）
```

**基礎給付（線形補間）**：年収200万→3.5、300→2.5、400→1.5、500→0.8、600→0.3、700→0.0（700万以上は0）。

- 思想：担税力の低い世帯（低所得・ひとり親・高齢）に厚く、高所得には出さない（**応能負担**）。
- **【査読対応⑤】加算の適用条件（誤適用防止）**：
  - **ひとり親加算（＋3.0）**：18歳以下／扶養中の子を持つ**単独養育世帯に限る**。子がいなければ適用しない（高所得の単身者がフラグONにしても加算されない）。所得制限を設ける場合は別途明記。
  - **高齢加算（＋1.0）**：高齢世帯＝**成人全員が65歳以上**の世帯に限定（§2）。三世代等の混在世帯には付けない（医療BSのみ高齢者本人の年齢区分で計上）。
- 実装整理：課税最低限以上は所得税からの**還付付き税額控除**、以下は**現金給付**として執行（マイナンバー連携・申請不要のプッシュ型）。シミュレーターは「給付相当額」を表示。
- 旧版指摘②への対応：EITC型を名乗りつつ実態が所得連動だった点を、**制度式として明示**（係数は政策仮定）。

## 6. ④ BS無償化の恩恵 ── 単価と出典（本モデルの核）

「無償化が肩代わりする**新規の**年間自己負担」を、世帯の構成から積み上げる。

```
④BS ＝ 教育BS（子の学齢別）＋ 医療BS（世帯員の年齢別合算）＋ 介護BS（要介護世帯）＋ 子育てBS（0〜2歳保育＋おむつ）
```

### 6-1. 教育BS 単価（万円/年・子1人）

| 区分 | 単価 | 根拠／タグ |
|---|--:|---|
| 小学生 | 7.0 | 文科省 子供の学習費調査(R5)を参考に 給食≈4＋学用品・教材≈3 を仮置き。**教科書は元々無償なので除外**【代表値】 |

## 7. ⑤ 社会保険料の軽減 ── 参考・トグル（既定OFF）【設計依存】

本改革の本質は「賃金にかかる社会保険料（現役偏重）→ 全世代の消費税」への組み替え。消費税（BS財源）の余裕を医療・介護の公費に投入すれば、**医療・介護分の保険料が下がる**。

```
保険料軽減 ＝ min( 年収 × 保険料率 × 0.20 , 上限5.0 )
保険料率：現役 5.5% ／ 高齢 6.0%   （年金保険料は別建て・動かさない）
```

- 財源は純増収の余裕（約5〜6兆）の一部（≈2〜3兆）。高所得は上限5万（応能負担）。
- **【査読対応⑦】この式（年収×保険料率×20%）は極めて粗い**。実際の軽減額は **加入する保険（被用者保険／国保／後期高齢者医療／介護保険）・所得・自治体・世帯構成により大きく異なり**、どれだけ下げられるかは保険者財政と制度設計次第。**本体試算には含めない**（参考のみ・既定OFF）。
- これを含めると「払う側」は ⑧高所得・子なし と ②単身中所得 に狭まる（§10）。**本体（消費税＋BSのみ）が保守的（現役の払う側に厳しめ）**である点の確認に用いる。

## 11. 正直な留保（レビューで明示確認いただきたい前提）

1. **静学・最終形**：世帯試算は消費減（動学）を世帯別に反映しない＝負担側に保守的。発動は凍結条件付きで、実際は段階的。
2. **代表値の粗さ**：消費係数（disp/prop/f_food）・BS単価（大学/介護/保育）・給付係数は代表値の仮置き。家賃/持ち家・地域・利用実態で変わる。**確定的な家計予測ではない**。
3. **医療・介護・保育の幅**：医療は年齢階級別平均（個人差大）。介護は在宅要介護2-3（施設はさらに大）。保育は認可利用の中間所得（自宅・幼稚園・認可外には効かない）。
4. **⑤保険料軽減は設計依存**：保険者財政の作り込みが要る方向性の参考。既定OFF。
5. **「保険」＝生涯の支え合い**：BS恩恵はライフステージ連動、税負担は恒常的。**一時点の損得でなく生涯で読む**（③+28.7は0〜2歳の間、⑦+14.7は子が大学の間の姿）。卒業・就学後は払う側へ、逆に今払う②⑧も将来は受け取る側へ移る。
6. **現実との別**：現実の政府は減税方向。本パッケージは**仮想内閣の思考実験**であり現実の税制ではない。
7. **BS費目内訳の残課題（§8-2）＝本仕様書は「財源枠内に収まることの確認」であり、BS費目ごとの確定積算ではない**。医療は年齢別人口で枠（≈2兆）と整合確認済み（高確度）。一方 **教育・介護・出産病児には約2.2兆円の未精査枠が残る**（給食全校無償化・高校以下の教材費・施設介護・出産費無償化等の対象人数×利用率×単価の精緻化は次版の課題）。この点は隠さず本文（§8）でも留保する。
8. **人口ウェイトは仮置き（§10-2）**：分布の純プラス/マイナス割合は、世帯類型・所得分布・子構成の代表値に依存する。確定的な人口推計ではなく、**向き（払う側＝子なし現役・高所得）の頑健性**を見るためのもの。
