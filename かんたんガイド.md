# 消費税改革インパクト・シミュレーター v3 ── かんたんガイド（使い方＋計算根拠）

**対象**：「暮らしの安心・消費税改革パッケージ」（標準16〜17%・食料品ゼロ税率・給付付き税額控除・ベーシックサービス無償化）が**フル実施されたとき、あなたの世帯の「年間の差し引き」**をざっくり試算するツールです。
**性質**：すべて粗い仮置きの**【仮想試算】**。確定値ではありません。これは「ひとつの党」AI仮想内閣プロジェクト（フィクション・思考実験）の試算で、現実の政府の政策ではありません。

---

## 1. これで何がわかる？

「消費税が上がると、うちは損？得？」── その問いに、**4つの増減を足し合わせて**ざっくり答えます。

```
年間の差し引き ＝ −① 増税の負担（非食料＋外食・酒類）
              ＋② 食料品ゼロ税率の恩恵
              ＋③ 給付付き税額控除（低所得・子育て・高齢などに）
              ＋④ ベーシックサービス無償化の恩恵（教育・医療・介護・保育）
```

**プラスなら「受け取る側」、マイナスなら「支える側」**です。大事なのは ──

> **全員が毎年「得」をする制度ではありません。** これは払った税を取り戻す「還付」ではなく、**社会全体の「保険」**。子育て・医療・介護など人生でお金が要る時期に厚く返り、余力のある時は支える側に回ります。**一時点の損得ではなく、生涯の支え合い**として見てください。

---

## 2. 使い方（3ステップ）

1. **税率を選ぶ**：17%（既定）か 16%。必要額から逆算した水準です。
2. **世帯のタイプと年収を入れる**：単身／夫婦のみ／子育て／ひとり親／高齢…と、税込年収。
3. **結果を見る**：大きな数字が「年間の差し引き」。下の棒グラフが①〜④の内訳です。

**「くわしい」タブ**では、大人それぞれの年齢、0〜2歳の認可保育利用、要介護の有無、社会保険料の軽減（参考）も指定できます。

---

## 3. 結果の読み方

- **大きな数字**＝年間の差し引き（プラス＝恩恵／マイナス＝負担）。
- **棒グラフ**＝①増税（負担・赤）／②食料ゼロ・③給付・④BS（恩恵・緑）。
- **分布の位置づけ**＝人口全体で見ると、標準17%で**約6割が純プラス・約4割が純マイナス**（中央値 +1.8万）。**負担は「高所得ほど」ではなく、子・大学生・介護などの受益がない世帯（子なし現役・共働き子なし）に集中**します。

---

## 4. 計算の根拠（やさしい版）

### ① 増税の負担／② 食料ゼロの恩恵
家計調査の消費支出を**税抜に直してから**税率差を掛けます（税込のまま掛けると過大計算になるため）。
- 軽減税率の対象は「飲食料品（**酒類を除く**）」で、**外食・ケータリングは対象外**（国税庁）。なので**外食・酒類は①の標準税率の負担側**に入れ、ゼロ税率の恩恵（②）は内食だけにしています。

### ③ 給付付き税額控除
担税力の低い世帯に厚く、高所得には出しません（応能負担）。
- 基礎給付（年収が低いほど大）＋低所得加算＋子ども加算＋ひとり親加算＋高齢加算。
- 申請不要のプッシュ型（マイナンバー連携）を想定。

### ④ ベーシックサービス無償化（「肩代わりされる新規の自己負担」）
| 区分 | 目安（万円/年） | 出典・考え方 |
|---|--:|---|
| 小学生 | 7 | 文科省 学習費調査（給食＋学用品。教科書は元々無償で除外） |
| 中学生 | 8 | 同上 |
| 高校生 | 0 | **2026年度〜 所得制限撤廃で授業料は既存制度がカバー**（新規恩恵なし） |
| 大学生 | 0〜27 | 国立授業料の半額相当を基準に、**既存の修学支援との差分（新規分）だけ**。低所得は既存で軽減済み＝小さい |
| 医療 | 0.9〜2.6/人 | 国民医療費（年齢階級別）×実効自己負担率×軽減率（年齢で個別計上） |
| 介護（在宅要介護2〜3） | 15 | 介護保険の自己負担軽減 |
| 0〜2歳保育（認可）＋おむつ | 33 | 自治体の認可保育料＋おむつ無料化（3〜5歳は既に無償） |

> いずれも控えめ側に置いています。**医療は年齢別人口で全国の枠（約2兆円）と整合**を確認済み。**全国のBS所要7.5兆円のうち、約2.2兆円は費目内訳がまだ粗い未精査枠**で、本ツールは「財源枠内に収まること」の確認であり、費目ごとの確定積算ではありません。

## 5. 使うときの注意（留保）

- **確定的な家計予測ではありません**。家賃／持ち家・地域・受診や通学・保育の利用実態で変わります。消費の係数は家計調査の代表値からの内挿で、任意入力は近似です。
- **静学**（消費が減る効果を世帯別には反映していない）＝負担側にやや厳しめ（保守的）。
- **発動は凍結条件つき**：物価が落ち着き、実質賃金のプラスが定着するまで発動しない設計。本ツールは「フル実施時」の姿です。
- **現実の政府は減税方向**。これは仮想内閣の思考実験です。
