# 消費税改革インパクト・シミュレーター（仮想試算）

「ひとつの党｜AI仮想内閣」プロジェクト（**フィクション・思考実験**）の家計負担シミュレーターです。
消費税改革パッケージ（標準 **16〜17%**・**食料品ゼロ税率**・**給付付き税額控除**・**ベーシックサービス無償化**）が、
あなたの世帯の家計にどう効くかを、年収・世帯構成から試算します。

🔗 公開URL：https://yamazakimakoto.github.io/hitotsu-tax-sim/

## これは何か

- 年間の家計差し引き ＝ **−①増税（非食料＋外食・酒類）＋②食料品ゼロ税率＋③給付付き税額控除＋④ベーシックサービス無償化**（＋⑤社会保険料軽減＝参考）
- 食料品でも酒類・外食は軽減対象外（国税庁）のため、ゼロ税率の恩恵から外し標準税率側に算入。
- 計算エンジン（`calc_household.py`）と同一ロジックを JavaScript に移植。10のモデル世帯で一致を検証済み。

## 重要な前提（免責）

- 本ツールは**フィクション・思考実験**です。現実の政府・与党は減税の方向で議論している局面であり、本提案はそれとは**異なる方向の構想**です。
- 統計（社会保障費・税収・国民医療費等）は公的一次資料に基づきますが、**税率16〜17%・配分・世帯別影響額はすべて【仮想試算】**です。
- 実在の政党・政治家・団体の意思決定を示すものではありません。
